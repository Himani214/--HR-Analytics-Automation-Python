# 📊 HR Analytics & Automation using Python

---

## 📌 Project Overview
**HR Analytics & Automation using Python** is a project that demonstrates how data analysis and automation can improve HR decision-making and streamline repetitive tasks.  

The project focuses on key HR functionalities such as:  
- **Resume Screening**  
- **Employee Attrition Analysis**  
- **Performance Tracking**  
- **Automated Email Generation**  
- **HR Dashboard Insights**  

---

## 🚀 Features

### 🔍 Resume Screening
- Filters and analyzes resumes based on predefined criteria  
- Helps shortlist candidates efficiently  

### 📉 Employee Attrition Analysis
- Identifies patterns behind employee turnover  
- Helps HR teams make data-driven retention strategies  

### 📊 HR Dashboard Summary
- Provides insights using data visualization  
- Summarizes employee data for quick decision-making  

### 📧 Automated Email Generation
- Generates emails for HR communication  
- Reduces manual effort in repetitive tasks  

### 📈 Performance Reports
- Tracks employee performance  
- Generates structured reports for evaluation
