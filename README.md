📊 HR Analytics & Automation using Python
📌 Project Overview
This project is a Python-based HR Analytics and Automation system. It demonstrates how data analysis and basic automation can improve HR decision-making and streamline repetitive tasks.
The project focuses on key HR functionalities such as:
Resume screening
Employee attrition analysis
Performance tracking
Automated email generation
HR dashboard insights

🚀 Features
🔍 Resume Screening
Filters and analyzes resumes based on predefined criteria
Helps shortlist candidates efficiently
📉 Employee Attrition Analysis
Identifies patterns behind employee turnover
Helps HR teams make data-driven retention strategies
📊 HR Dashboard Summary
Provides insights using data visualization
Summarizes employee data for quick decision-making
📧 Automated Email Generation
Generates emails for HR communication
Reduces manual effort in repetitive tasks
📈 Performance Reports
Tracks employee performance
Generates structured reports for evaluation

🛠️ Tech Stack
Python
Pandas – Data manipulation & analysis
NumPy – Numerical computations
Matplotlib / Seaborn – Data visualization

🧰 Tools & Platforms
Python (Pandas, NumPy) – For data analysis and processing
Matplotlib / Seaborn – For data visualization

⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/your-username/HR-Analytics-Automation-Python.git

Navigate to the project folder:
cd HR-Analytics-Automation-Python

Install required libraries:
pip install pandas numpy matplotlib seaborn

Open and run the project file (week_1_lab_1.ipynb) in your preferred environment.
